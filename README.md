## System requirements:
* Installed Node.js version 13 or higher

## Install project:
1. Clone repository using the command:
```
git@github.com:super-gr0ver/hexlabs.git
```
2. Enter to project directory:
```
cd hexlabs
```
3. Install npm packaje using the command:
```
npm install
```
